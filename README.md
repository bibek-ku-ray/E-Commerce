# Ecommerce MERN Application

# How to run project

- go to client and run command: `npm i` and `npm run dev`
- go to server and run command: `npm i` and `npm run dev`


# Stack 

- React
- Mongodb
- Express

# Sub-Stack

## Backend

- bcryptjs

- cors
- dotenv
- jsonwebtoken
- mongoose
- multer
